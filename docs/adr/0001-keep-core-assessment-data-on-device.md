# Keep Core Assessment Data On-Device

AccessiRoom stores Captured Rooms, Mobility Profiles, and confirmed Room Setups on-device because they contain sensitive household geometry and personal mobility needs, and the core assessment loop does not require cloud processing. Deterministic assessment results are derived on demand from those persisted inputs rather than stored as a second source of truth; an explicitly exported Assessment Report is their durable record. Data leaves the device only through explicit Operator export, and future synchronization, hosted analysis, or assistant features must preserve this boundary or deliberately supersede this decision.
